Analysing and Visualising the Dataset of Techno Film from Super8database¶

In this notebook I will analyza a simple dataset I downloaded from Super8database.org to visualize some of the genres that have been distributed by the Italian distributor Techno Film during the 70s and 80s. The company sold material not only in Italy, but also in Europe, in Spain or in the Netherlands. They were so widespread that some of those small gauge films are very easy to find even today on ebay.

import pandas as pd
import numpy as np

# Your DataFrame is assumed to be already loaded
# technofilm_df = pd.read_csv("yourfile.csv")  # or however you loaded it

# Define the function
def assign_genre(tags):
    if pd.isnull(tags):
        return np.nan  # or "Unknown"
    tags = tags.lower()  # lowercase everything
    if "cartoon" in tags:
        return "cartoon"
    elif "documentary" in tags:
        return "documentary"
    elif "sci-fi" in tags:
        return "sci-fi"
    elif "western" in tags:
        return "western"
    elif "adventure" in tags:
        return "adventure/action"
    elif "action" in tags:
        return "adventure/action"
    elif "drama" in tags:
        return "drama"
    else:
        return "Others"

# Apply the function
technofilm_df['genre'] = technofilm_df['tags'].apply(assign_genre)

#counting the frequency of genre per film 

genre_list = pd.DataFrame(technofilm_df.genre.value_counts())
genre_list["Frequency"] = technofilm_df.genre.value_counts(normalize=True)
genre_list = genre_list.reset_index()
genre_list = genre_list.rename(columns = {"index": "genre", "count": "Number of Films"})
genre_list.loc[genre_list.Frequency < 0.03, 'genre'] = 'Others'
genre_list = (genre_list
                .groupby('genre').sum()
                .sort_values('Number of Films', ascending=False)
                .reset_index())
print(genre_list)
